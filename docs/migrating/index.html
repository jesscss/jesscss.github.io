Migrating to Jess

Variable Declaration#

Variables are declared using the @let at-rule.

Less

@color: red;

Sass

$color: red;

Jess

@let color: red;

Variable Reference#

Jess can evaluate any continuous JavaScript expression starting with $.

Less

color: @color;

Sass

color: $color;

Jess

color: $color;

Math#

Depending on the type of value, and if you want to preserve units, you can use Jess helper functions (coming soon). Otherwise, you can just perform your math in JavaScript.

Less

width: (@width * 2);

Sass

width: ($width * 2);

Jess

// Note: this will only work for primitive values.
// This is just a plain JS expression!
width: $(width * 2);

or (not available in alpha yet)

@import { multiply } from '@jesscss/fns';
width: $multiply(width, 2);

Mixins#

Less

.mixin(@size) {
width: @size * 2px;
}
.box {
.mixin(50);
}

Sass

@mixin mixin($size) {
width: $size * 2px;
}
.box {
@include mixin(50);
}

Jess

@import { multiply } from '@jesscss/fns';
@mixin mixin(size) {
width: multiply($size, 2px);
}
.box {
@include mixin(50);
}

Each#

Sass

@each $size in $sizes {
.icon-#{$size} {
font-size: $size;
height: $size;
width: $size;
}
}

Less

each(@sizes, #(@size) {
.icon-@{size} {
font-size: @size;
height: @size;
width: @size;
}
})

Jess

Note: not available yet in alpha, but will work like the following:

@import { each } from '@jesscss/fns';
@let sizes: 24px, 32px, 40px;
@mixin size(size) {
.icon-$(size) {
font-size: $size;
height: $size;
width: $size;
}
}
@include each($sizes, $size);
